# Choice Medical

Website for Choice Medical — investor-facing site for a seed-stage medical device startup developing the first fully biodegradable breast biopsy marker.

## Tech stack

- React 18 + TypeScript
- Vite 7
- Tailwind CSS
- shadcn/ui

## Development

```sh
npm install
npm run dev
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on every push to `main`.
